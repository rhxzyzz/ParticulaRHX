# ParticulaRHX

**Partículas Animadas que Formam um Coração e um X**

Um projeto visual interativo que usa partículas para criar formas de coração e X com animações fluidas e efeitos visuais interessantes.

## 🎨 Características

- ✨ **Animação de Partículas**: Partículas que se movem em direção a pontos-alvo para formar as formas
- ❤️ **Forma de Coração**: Partículas rosa/vermelho formando um coração
- ❌ **Forma de X**: Partículas azul/roxo formando um X
- 🎮 **Controle Interativo**: Botão Play/Pause para iniciar/pausar a animação
- 🌐 **Canvas HTML5**: Renderização suave com efeitos de conexão entre partículas
- 📊 **Contador**: Exibe o número total de partículas em tempo real

## 🚀 Como Usar

1. Abra `index.html` em um navegador web
2. Clique no botão **Play** para iniciar a animação
3. Observe as partículas se organizarem em formas de coração e X
4. Clique em **Pause** para pausar a animação a qualquer momento

## 🛠️ Tecnologias

- **HTML5**: Estrutura da página
- **Canvas 2D**: Renderização gráfica
- **JavaScript**: Lógica de partículas e animações
- **CSS3**: Estilização e efeitos visuais

## 📦 Arquivos

- `index.html` - Página principal com canvas
- `script.js` - Lógica de partículas e animações
- `README.md` - Este arquivo

## 🎯 Características Técnicas

- Sistema de partículas baseado em física (forças de atração)
- Amortecimento (damping) para movimento suave
- Geração automática de pontos em forma de curva paramétrica (coração)
- Geração de pontos para forma de X
- Efeito de linha de conexão entre partículas próximas
- Cores dinâmicas e opacidade baseada na distância
